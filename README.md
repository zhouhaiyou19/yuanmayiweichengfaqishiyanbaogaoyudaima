1. **实验报告**：详细记录了实验的背景、目的、实验步骤、实验结果及分析等内容。报告内容由本人亲自撰写，确保内容的准确性和可读性。

通过本次实验，您将能够：

1. 掌握原码一位乘法运算的基本原理。
2. 熟练掌握Logisim寄存器电路的使用。
3. 在Logisim平台中设计并实现一个8*8位的无符号数乘法器。
